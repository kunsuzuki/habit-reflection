プロジェクト概要：
AIハビットトラッカー
（AI × 健康管理　× 日記　× TODO　× 習慣トラッカー）

「マイ ルーティン」というスマホアプリはご存知ですか？
こちらのプロジェクトは、そのアプリに、日記テキスト入力やAI分析を組み込んだものだというイメージを持っていてください。

主な機能：
ユーザーは、チェックリスト形式で毎日の習慣トラッカーを作成できる（朝、昼、夜OR時間選択可能）
ユーザーは、達成したらチェックを行い日々の習慣化したいものを可視化できる
さらに、日記（テキストエリア）にて、1日の中で、考えたことや振り返り、感情、次の日の目標など自由に記載して保存できる機能がある
1日の最後（全ての習慣チェックが完了、日記入力が完了したタイミング）にAIによる振り返りボタンを押すことで、
AIがその日の習慣の進捗度や日記に書かれた内容を分析して、ユーザーの健康状態やメンタル状態、モチベーションを分析してくれる。分析した内容をもとに、次の日から取るべき行動や、見直すべき習慣などをアドバイスしてくれる。
→AIのAPIは、収益が発生するまで、またはMVPの段階では、無料APIを使用する（Gemini API？）
AIが提案した推奨TODOを、次の日の予定に組み込むことができる。（次の日の予定に組み込みますか？Yes/No）
有料プラン：習慣トラッカーの高度なカスタム化（時間設定、曜日設定など）
AI分析の拡大・精度高い分析提供（過去3日間、1週間の内容からさらに深く踏み込んだ分析）

簡単な画面設計：
一つの画面に、習慣管理、TODO、日記テキスト入力、AI分析が表示されるイメージ
習慣管理とTODOはタブで切り替えを行うようにする
日記テキスト部分、AI分析部分は、タブに下に配置（どちらのタブでも閲覧可能）
左にスライドすることで、前の日付の内容に移動、右にスライドすることで、次の日付の内容に移動
画面上部（ヘッダー下あたり）に、横一列で日付が表示されている（当日の日付は分かりやすく強調、色を変えるとか）
日付を選択することで、その日付に移動
日付が並んでる上に、その月が表示されている
月をクリックすると、カレンダー画面に遷移


技術スタック（最適化版）：
フロントエンド：Next.js, Tailwind CSS
バックエンド：Next.js API Routes + Cloudflare Workers
DB：Cloudflare D1（無料枠が十分、Cloudflareエコシステム内）
API：Gemini API + Cloudflare Workers AI（バックアップとして）
ホスティング：Cloudflare Pages + Workers

データモデル：
- User: ユーザー情報
- Habit: 習慣定義（名前、説明、頻度、時間帯など）
- HabitLog: 習慣の実行記録（日付、完了状態）
- Journal: 日記エントリ（日付、テキスト、感情タグなど）
- Todo: TODOアイテム（タイトル、説明、期限、優先度など）
- AIAnalysis: AI分析結果（日付、分析テキスト、推奨アクション）

開発工程：MVPは、スマホ画面のみ
1）基本UI/UXフレームワーク（カレンダー、日付ナビゲーション）
2）習慣トラッカー基本機能
3）日記機能
4）AI分析（最小限の機能から）
5）TODO機能
6）ユーザー認証（最初からでも良いですが、機能検証を優先する場合は後回しも可）

※習慣トラッカーとTODOの違いは、習慣トラッカーは習慣化したいタスク（繰り返したいタスクを登録）TODOは、その日限りのタスクやその日にチャレンジするタスクを記録。

AI機能の段階的実装：
1. 基本感情分析（ポジティブ/ネガティブ/ニュートラル）
2. キーワード抽出と傾向分析
3. 習慣達成パターンとの相関分析
4. パーソナライズされた提案生成

収益化モデル：
- 基本機能：無料
- プレミアム機能：
  - 高度なAI分析（長期トレンド、詳細な健康アドバイス）
  - 習慣テンプレート（健康、生産性、学習など）
  - データエクスポート機能
  - 詳細な統計とレポート
